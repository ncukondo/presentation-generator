# Layouts

再利用可能なスライドレイアウト関数。各 `pages/*.ts` はここから import して組み合わせる。

デザイン原則は `docs/design-principles.md` を参照。新しいレイアウトを追加する時は:

1. 原則「1スライドに色3つ以下」「最小フォント22pt」「余白 0.5in」を満たすデフォルト値にする
2. 引数は少なく、セマンティックに意味のあるフィールド名で受ける
3. 色は step/random ではなく `C.primary` か引数指定

## 現在のレイアウト一覧

| ファイル | 用途 | 使用例 |
|---------|------|--------|
| `hero-quote.ts` | 1スライド全面の大きな引用・宣言 | 章扉・締めの一言 |
| `number-card-grid.ts` | 番号付き並列カード（2-4カード） | 背景の課題・準備項目 |

## 追加予定

- `half-bleed-image.ts` — 画面半分に画像、半分にテキスト
- `data-grid.ts` — 均質な 2x2 / 3x2 項目グリッド
- `timeline.ts` — 時系列（3-6ステップ）
